## When to use WebFetch

Fetch a doc page when the user asks about:
- Pricing or billing details for a resource
- Resource limits (max nodes, max volume size, etc.)
- Feature availability by location (Delhi vs Chennai)
- Configuration options not listed in the reference file
- Error codes from the MyAccount API
- Release notes for a specific version

Fetch only the specific page needed. Do not fetch the full docs site.
